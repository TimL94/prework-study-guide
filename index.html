HTML

  • The head contains meta data that is useful for assitive technologies and search engines.
  • The body contains elements that the user will see and interact with on the webpage.
  • You can use inline, internal and external CSS style to style your web page.
  • The external style is the most commonly used because it allows more control.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A comma can be used to create rules for multiple elements.
  • You can use inline, internal and external CSS style to style your web page.
  • The external style is the most commonly used because it allows more control.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the current main branch
  • git commit: commits changes to the main branch

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Javascript allows the user to interact with your web application!
